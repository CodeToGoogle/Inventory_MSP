# Auth Service

This service is responsible for user authentication and authorization.

## Environment Variables

| ENV Variable      | Purpose                |
|-------------------|------------------------|
| `AUTH_DB_URL`     | DB URL                 |
| `AUTH_DB_USERNAME`| DB user                |
| `AUTH_DB_PASSWORD`| DB password            |
| `EUREKA_URI`      | Eureka server URL      |
| `JWT_SECRET`      | JWT Signing secret     |
| `JWT_EXPIRATION`  | Token expiry           |
| `KAFKA_BOOTSTRAP` | Kafka cluster address  |
